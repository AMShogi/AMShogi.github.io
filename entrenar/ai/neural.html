Neural Network engine AobaZero

  • Download CPU exe and download Latest weight=
  • In the folder aobazero-XX-w64-cpu make a new folder named weight-save and inside it copy-paste the extracted weight.
  • Right-click aobaz.bat and select edit. It'll open in Notepad and change the bold numbers of the last line:
    • bin\aobaz -q -i -p 100 -h 1 -w weight-save\w000000002184.txt
    • To match the number of w00000000#### .txt file in your weight-save folder.
  • If instructions are unclear AobaZero has an English README.md
  • To increase depth right-click aobaz.bat and select edit. It'll open in Notepad.
    • Increase by 100 the bold number.
    • bin\aobaz -q -i -p 100 -h 1 -w weight-save\w00000000####.txt
  • Go to Game > Engine Manager > Add and select aobaz.bat

How to Analyze A Game?

After finishing a game in 81dojo, click these to copy the KIFU notation text.

Go back to Shogidokoro, then click Edit > Paste Game (or simply press Ctrl+V).

  • Go to Game(G) > Analyze Game and click Set and Start

How the engine evaluates the position. A positive number means Black is better, while a negative number means White is better.

  1. A value of ±1000 or larger means that one side has a clear advantage.
  2. Condition: how the engine evaluates the position, but in words.

Move eval: how the engine evaluates the move.

  1. An exclamation point (!) means the engine thinks the move is actually better than its own suggestion.
  2. A large circle (O) means the engine agrees with the move.
  3. A dot (.) means the move is "inferior", but still very decent.
  4. Finally, in order of increasingly worse evaluations:
    1. ?
    2. Dubious move ?
    3. Bad move ??
  1. Diff: the jump in value from the previous position.
  2. Candidate: engine's suggested move.
  3. Moves: the sequence of moves that engine sees as best for both sides.

Tips on using an engine

  1. As a beginner, focus more on large spikes in the evaluation. If you are Sente, and you played a move that reduces the evaluation from 200 to -200, relax. It's still even. But a drop from 200 to -1000 is worth looking into.
